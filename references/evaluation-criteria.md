# 评标标准库

## 常见评标方法

### 综合评分法（最常用）
```
综合得分 = 技术得分 × 技术权重(60-80%) + 价格得分 × 价格权重(20-40%)
```

### 最低价评标法
- 满足招标文件实质性要求后，价格最低者中标
- 适用于标准化程度高的货物采购

### 性价比法
```
得分 = 技术得分 / 投标报价
```

## 技术标评审维度

### 1. 技术方案完整性 (20-30分)
- 需求理解与响应
- 方案设计逻辑性
- 实施方法可行性

### 2. 企业资质与业绩 (15-25分)
- 类似项目经验
- 资质证书等级
- 团队配置

### 3. 质量保证措施 (10-15分)
- 质量管理体系
- 验收标准明确性
- 售后服务承诺

### 4. 工期与进度 (10-15分)
- 进度计划合理性
- 里程碑设置
- 风险预案

### 5. 报价合理性 (20-40分)
- 总价合理性
- 分项报价透明性
- 成本结构清晰度

## 价格分计算示例

假设：基准价 = 所有有效投标报价的最低值

| 报价方式 | 计算公式 |
|---------|---------|
| 最低价法 | 价格分 = (基准价 / 投标价) × 满分 |
| 中间价法 | 价格分 = (1 - \|投标价-基准价\|/基准价) × 满分 |
| 平均值法 | 价格分 = (基准价 / 投标价) × 满分，基准价=最低报价 |

## 废标条款（常见）

1. 投标文件未密封/盖章
2. 投标报价超过预算
3. 资格证明文件缺失或过期
4. 技术方案不满足实质性要求
5. 投标截止时间后递交
6. 联合体协议缺失或不符合要求
