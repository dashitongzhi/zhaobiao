# 投标策略参考

## 1. 竞争策略选择

### 低价抢标策略
适用场景：
- 进入新市场需要业绩背书
- 技术门槛低、竞争充分
- 我方有成本优势

风险：可能陷入价格战，压缩利润空间

### 差异化策略
适用场景：
- 技术方案有独特优势
- 客户对质量敏感度高
- 竞争对手同质化严重

关键：必须在技术方案中清晰传达差异化价值

### 均衡策略
适用场景：
- 成熟项目，竞争格局稳定
- 客户预算明确，降价空间有限
- 评标中价格分权重适中（30-50%）

## 2. 评分博弈模型

评标总分 = 技术分 × 技术权重 + 价格分 × 价格权重

**价格分计算方式**：
- 最低价法：最低价得分 = 满分
- 中间价法：接近平均价得分最高
- 加权平均法：低于平均价适当加分

**策略建议**：
1. 测算竞争对手成本线，确定心理底价
2. 根据价格权重调整报价策略
3. 技术分领先时，报价可适度上浮

## 3. 投标窗口期管理

| 距离截止 | 行动 |
|---------|------|
| 30天+ | 充足准备期，重点打磨技术方案 |
| 14-30天 | 标准准备，商务报价精细化 |
| 7-14天 | 加速准备，检查资质完整性 |
| <7天 | 应急模式，优先保证必选项 |
| <48小时 | 仅建议已有成熟方案时参与 |

## 4. 资格预审应对

- 仔细核对资格要求，标注缺失项
- 寻找合作方弥补资质缺口（联合体投标）
- 关注是否接受电子化资质证明
- 预留资质办理时间（通常需要15-30天）
