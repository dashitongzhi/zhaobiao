## 合同关键条款检查清单

### 付款条款
- [ ] 预付款比例（通常10-30%）
- [ ] 进度款节点设置
- [ ] 验收款比例（通常50-60%）
- [ ] 质保金比例（通常5-10%）
- [ ] 付款周期（30/60/90天）

### 履约条款
- [ ] 履约保证金金额与退还条件
- [ ] 履约期限与延期罚款
- [ ] 验收标准与方法
- [ ] 质量问题处理流程

### 风险条款
- [ ] 不可抗力定义与处理
- [ ] 违约责任界定
- [ ] 知识产权归属
- [ ] 保密条款范围
- [ ] 争议解决机制（仲裁/诉讼）

## 风险预警红线

⚠️ 高风险条款：
1. 验收标准模糊，无法量化
2. 付款条件过于苛刻（验收后才付90%+）
3. 质保期过长（超过24个月）
4. 违约责任不对等
5. 知识产权归属不清晰
