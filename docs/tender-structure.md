# 招标文件标准结构研究报告

> 基于《中华人民共和国招标投标法》《中华人民共和国政府采购法》及相关法规要求

---

## 一、资格要求章节（资格审查）

### 1.1 法律依据
- 《招标投标法》第十八条：招标人应当根据招标项目的特点和需要编制招标文件。招标人不得以不合理条件限制或者排斥潜在投标人，不得对潜在投标人实行歧视待遇。
- 《政府采购法》第二十二条：供应商参加政府采购活动应当具备特定条件

### 1.2 资格条件内容

#### 基本资格条件
1. **法人资格**：具有独立承担民事责任的能力（有效营业执照、税务登记证、组织机构代码证）
2. **财务状况**：具有良好的商业信誉和健全的财务会计制度
3. **纳税社保**：有依法缴纳税收和社会保障资金的良好记录
4. **经营年限**：参加政府采购活动前三年内，在经营活动中没有重大违法记录
5. **信用记录**：通过"信用中国"网站等渠道查询，无不良信用记录

#### 专业资质条件（根据项目需要）
1. 企业资质等级（建筑业企业资质、安全生产许可证等）
2. 相关行业许可证（通信工程、安防工程等）
3. 质量管理体系认证（ISO9001等）
4. 环境管理体系认证（ISO14001等）
5. 特定人员执业资格（项目经理、注册工程师等）

### 1.3 资格审查方式

| 方式 | 适用场景 | 说明 |
|------|----------|------|
| 资格预审 | 大型复杂项目、技术复杂项目 | 投标前审查，筛选合格投标人 |
| 资格后审 | 一般项目 | 开标后评审，资格作为评分因素之一 |

### 1.4 禁止性要求
- 不得以注册资本、资产总额、营业收入等条件作为资格要求
- 不得将特定区域业绩作为资格要求
- 不得要求设立分支机构等不合理条件
- 不得以不合理方式限制或排斥潜在投标人

## 二、评分办法章节（评标办法）

### 2.1 法律依据
- 《招标投标法》第四十条：评标委员会应当按照招标文件确定的评标标准和方法，对投标文件进行评审和比较
- 《政府采购法实施条例》第三十四条：政府采购招标评标方法分为最低评标价法和综合评分法

### 2.2 评标方法类型

#### 最低评标价法
- **适用情形**：技术、服务等标准统一的货物采购项目
- **操作方式**：通过资格性审查和符合性审查后，以最低报价为成交候选人
- **优点**：简单透明，防止人为干预

#### 综合评分法（最常用）
- **适用情形**：技术复杂或特殊性服务项目
- **评分结构**：

| 评分因素 | 权重范围 | 说明 |
|----------|----------|------|
| 价格部分 | 30%-60% | 满足招标文件要求且价格最低 |
| 技术部分 | 40%-60% | 技术方案、设备配置、技术指标 |
| 商务部分 | 10%-20% | 业绩、信誉、售后服务 |
| 其他部分 | 5%-10% | 绿色环保、专利等 |

#### 综合评分法计算公式
```
综合得分 = 技术得分 + 商务得分 + 价格得分

价格得分 =（评标基准价 / 投标报价）× 价格权重 × 100

评标基准价确定方式：
- 最低价基准：以最低报价为基准
- 平均值基准：以平均报价为基准
- 最低有效报价基准：去掉最高和最低后的平均
```

### 2.3 评分细则要求

1. **评分标准应明确**
   - 每项评分因素的分值设置
   - 评分标准的具体依据
   - 得分档次划分（如优90-100分、良80-89分、中60-79分）

2. **量化打分要求**
   - 技术方案应有明确的评分标准
   - 避免评审专家主观性过强
   - 允许适度主观判断但需有据可查

3. **价格分计算方法**
   ```公式
   价格得分 = (最低报价 / 投标报价) × 价格权重分值
   
   或
   
   价格得分 = [1 - |投标报价-基准价|/基准价] × 价格权重分值
   ```

### 2.4 评分办法章节结构示例

```
第三章 评标办法

第一条 评标方法
本次招标采用 综合评分法 进行评标。

第二条 评标委员会组成
- 评标委员会由5人（单数）组成
- 其中招标人代表不超过1人
- 技术、经济专家不少于成员总数的2/3

第三条 评分细则

| 评分因素 | 权重 | 评分标准 |
|----------|------|----------|
| 价格部分 | 30分 | 见第四条 |
| 技术部分 | 50分 | 见第五条 |
| 商务部分 | 20分 | 见第六条 |
| 合计 | 100分 | - |

第四条 价格评分（30分）
（一）投标报价有效性核查...
（二）价格计算公式...

第五条 技术评分（50分）
（一）技术方案完整性（20分）...
（二）技术指标响应（20分）...
（三）项目实施计划（10分）...

第六条 商务评分（20分）
（一）类似项目业绩（10分）...
（二）售后服务方案（5分）...
（三）企业资质信誉（5分）...
```

## 三、技术规格章节（技术要求）

### 3.1 法律依据
- 《招标投标法》第十九条：招标文件应当包括招标项目的技术、质量要求
- 《政府采购法》第三十六条：招标文件中没有规定的评标标准不得作为评审依据

### 3.2 技术规格内容要求

#### 基本技术要求
1. **项目概述**
   - 项目背景与目标
   - 建设规模与范围
   - 总体技术路线

2. **功能需求**
   - 核心功能模块
   - 接口要求
   - 数据处理能力
   - 系统性能指标

3. **技术指标**
   - 硬件规格（设备型号、配置参数）
   - 软件规格（操作系统、数据库、中间件）
   - 网络要求（带宽、延迟、安全）
   - 环境要求（温度、湿度、机房条件）

4. **建设要求**
   - 工期要求
   - 验收标准
   - 质量保证
   - 培训要求

### 3.3 技术规格编写原则

| 原则 | 说明 | 示例 |
|------|------|------|
| 完整性 | 技术要求应覆盖所有关键指标 | 不能遗漏核心功能 |
| 明确性 | 指标应可量化、可验证 | "响应时间<2秒"而非"响应快" |
| 公正性 | 不得指定特定品牌或供应商 | 避免"须采用XX品牌" |
| 适度性 | 技术要求与项目实际需要匹配 | 不得超规格要求 |

### 3.4 常见技术规格问题规避

**禁止出现的表述：**
- ❌ "须采用XXX品牌产品"
- ❌ "须通过XXX认证"
- ❌ "指定特定供应商"
- ❌ 与实际需求不匹配的高规格要求

**推荐表述方式：**
- ✅ "产品应符合国家标准GB/T XXXX"
- ✅ "同等或以上规格的产品"
- ✅ "具有相关认证资质的企业"

## 四、合同条款章节

### 4.1 法律依据
- 《招标投标法》第四十六条：招标人和中标人应当自中标通知书发出之日起三十日内，按照招标文件和中标人的投标文件订立书面合同
- 《民法典》合同编：合同是当事人之间设立、变更、终止民事法律关系的协议

### 4.2 合同条款主要内容

#### 合同通用条款
1. **合同双方**
   - 甲方（采购人/招标人）信息
   - 乙方（中标人）信息

2. **合同标的**
   - 项目名称与编号
   - 采购内容与范围
   - 合同金额（含税率）

3. **权利与义务**
   - 甲方的权利义务
   - 乙方的权利义务

4. **合同工期**
   - 开工条件
   - 工期要求
   - 进度节点
   - 延误责任

5. **质量标准**
   - 质量要求
   - 验收标准
   - 质量保证期
   - 质量保证金

6. **付款条件**
   - 付款方式（预付款、进度款、验收款、质保金）
   - 付款节点
   - 发票要求

7. **违约责任**
   - 甲方违约责任
   - 乙方违约责任
   - 违约金计算

8. **争议解决**
   - 协商解决
   - 仲裁/诉讼管辖

### 4.3 专用合同条款（行业/项目特定）

```
示例：信息系统建设项目专用条款

第十条 项目验收
10.1 初验：系统部署完成后15日内完成初验
10.2 试运行：初验合格后试运行30日
10.3终验：试运行正常后组织终验

第十一条 售后服务
11.1 质保期：验收合格后24个月
11.2 响应时间：一般问题4小时，重大问题2小时
11.3 服务方式：7×24小时热线支持
```

### 4.4 合同特别条款

| 条款类型 | 内容要求 |
|----------|----------|
| 知识产权条款 | 明确软件著作权、源代码归属 |
| 保密条款 | 双方保密义务、数据保护 |
| 不可抗力 | 不可抗力定义与处理方式 |
| 合同变更 | 变更流程、价格调整机制 |
| 终止条款 | 提前终止条件、赔偿方式 |

## 五、时间节点安排

### 5.1 法律依据
- 《招标投标法》第二十四条：招标人应当确定投标人编制投标文件所需要的合理时间
- 《政府采购法》第三十五条：货物和服务项目实行招标方式采购的，自招标文件开始发出之日起至投标人提交投标文件截止之日止，最短不得少于二十日

### 5.2 标准时间节点

#### 公开招标时间节点（通用）

| 阶段 | 内容 | 时间要求 | 说明 |
|------|------|----------|------|
| 公告发布 | 发布招标公告 | T日 | 至少20日前公告 |
| 报名时间 | 潜在投标人获取招标文件 | T+1至T+N | 通常5-10日 |
| 现场考察 | 组织潜在投标人现场踏勘 | T+3 | 可选，招标人组织 |
| 疑问提交 | 潜在投标人提出疑问 | T+5至T+7 | 截止时间前 |
| 答疑澄清 | 发布答疑及澄清文件 | T+7至T+10 | 澄清修改构成招标文件 |
| 投标截止 | 投标人提交投标文件 | T+N | 不少于20日 |
| 开标时间 | 公开开标 | T+N | 同一时间 |
| 评标时间 | 评审委员会评审 | T+N+1 | 通常1-3日 |
| 结果公示 | 公告中标结果 | T+N+3 | 公示期3个工作日 |
| 中标通知 | 发放中标通知书 | 公示期满后 | 法定异议期后 |

#### 政府采购时间要求

| 情形 | 最短时限 | 依据 |
|------|----------|------|
| 公开招标 | ≥20日 | 政府采购法35条 |
| 竞争性谈判 | ≥3个工作日 | 谈判文件发出至响应文件截止 |
| 竞争性磋商 | ≥10日 | 磋商文件发出至响应文件截止 |
| 单一来源 | 公示期≥5个工作日 | 采购法39条 |

### 5.3 时间节点图示

```
T-20 ───────────────────────────────────────── T=0
 │                                             │
 ▼                                             ▼
招标公告发布                              投标截止/开标
 │                                             │
 ▼                                             │
T-15: 开始发售招标文件                       T+N: 评标
 │                                             │
 ▼                                             │
T-7: 疑问截止                               T+N+3: 结果公示
 │                                             │
 ▼                                             │
T-3: 澄清答疑发布                           T+N+7: 发放中标通知
```

### 5.4 关键时间点控制要点

1. **公告发布时间**：确保20日前发布，留足缓冲
2. **报名时间**：合理设置，建议5个工作日
3. **疑问时间**：给潜在投标人充足时间提出问题
4. **澄清发布**：至少提前3个工作日发布澄清
5. **投标截止**：严格执行"一剑封喉"原则
6. **结果公示**：及时公示，公示期不得缩短

## 六、招标文件整体结构目录

基于法规要求的完整招标文件结构：

```
第一章 招标公告/投标邀请书
 ├── 招标条件
 ├── 项目概况与招标范围
 ├── 投标人资格要求
 ├── 招标文件获取
 ├── 投标文件递交
 └── 联系方式

第二章 投标人须知
 ├── 前附表
 ├── 总则（项目概况、资金来源、投标人资格等）
 ├── 招标文件（组成、澄清、修改）
 ├── 投标文件（内容、格式、密封、递交）
 ├── 投标保证金
 ├── 开标与评标
 ├── 合同授予
 └── 纪律与监督

第三章 评标办法
 ├── 评标方法（最低价法/综合评分法）
 ├── 评标委员会
 ├── 评标程序
 ├── 评分细则（价格/技术/商务）
 └── 评标结果

第四章 采购需求/技术规格
 ├── 项目概述
 ├── 采购内容
 ├── 技术要求
 ├── 建设/实施要求
 ├── 验收标准
 └── 踏勘安排（如有）

第五章 合同条款
 ├── 合同协议书
 ├── 通用合同条款
 ├── 专用合同条款
 └── 合同格式附件

第六章 投标文件格式
 ├── 投标函
 ├── 法定代表人身份证明
 ├── 授权委托书
 ├── 资格证明材料
 ├── 投标保证金
 ├── 技术/商务响应表
 ├── 报价清单
 └── 其他材料

附件：
 ├── 踏勘登记表
 ├── 质疑函格式
 ├── 投诉书格式
 └── 廉洁承诺书
```

## 八、特别说明

### 政府采购与工程建设招标的区别

| 维度 | 政府采购 | 工程建设招标 |
|------|----------|--------------|
| 主管法规 | 政府采购法 | 招标投标法 |
| 监管体系 | 财政部门 | 发展改革部门/行业主管部门 |
| 质疑投诉 | 向采购人或采购代理机构提出 | 向招标人或行政监督部门提出 |
| 评审专家 | 政府采购评审专家库 | 综合评标专家库 |

### 电子招标特殊要求
- 应符合《电子招标投标办法》要求
- 招标公告、招标文件电子化
- 网上开标、评标
- 数据电文具有法律效力
